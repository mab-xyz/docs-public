# docs-public

Public documentation for [mab.xyz](https://mab.xyz) products.

mab.xyz builds security infrastructure for Ethereum. Our APIs let wallets, frontends, and compliance workflows assess transaction risk before broadcast — flagging unverified contracts, first-time interactions, and other on-chain danger signals in real time.

## Contents

- [api.mab.xyz.md](api.mab.xyz.md) — REST API reference for `api.mab.xyz`, including the `/v1/analysis/tx-risk-raw` endpoint with full request/response documentation and runnable curl examples.
